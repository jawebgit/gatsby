# gatsby-plugin-jss

Provide drop-in support for using the css-in-js library
[JSS](https://github.com/cssinjs/react-jss) including server rendering.

## Install

`npm install --save gatsby-plugin-jss`

## How to use

First add the plugin to your `gatsby-config.js`.

```javascript
plugins: [
  `gatsby-plugin-jss`,
]
```
